# FDC-client
> 房产中介小程序前端
> 
> 后端链接： [房地产中介小程序 GO语言 后端](https://github.com/Locter9001/FDC-server)
> 
> 前端链接： [房地产中介小程序 前端](https://github.com/Locter9001/FDC-client)
## 技术栈
    前端： 原生小程序 + vant UI
    后端： Go + Gin + Redis

## 制作背景
    该小程序适用于房地产相关从业者，其目的是为了更好的管理资源。
    操作流程： 员工在小程序上报备客户 领导审核 通过后 通知带班人员引导客户去看房产
